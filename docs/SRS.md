## 1.2 Бизнес-требования

<a name="initial_data"/>

### 1.2.1 Исходные данные
Все хотят держать себя в форме, однако не всем хватает мотивации для начала или продолжения поддержки себя в форме. Иногда необходимо, чтобы кто-то (или что-то) напоминали про физическую активность.

<a name="business_opportunities"/>

### 1.2.2 Возможности бизнеса
Большое количество людей оценило бы приложение, которое позволит им  установить интересные пользователю цели. Для мотивации пользователей можно использовать оповещения, тем самым напоминая пользователям про ранее установленные цели. Подобное приложение позволит все это осуществить. Благодаря интуитивно понятному интерфейсу, приложением сможет пользоваться любой человек.


<a name="analogues"/>

## 1.3 Аналоги
Fitbit предлагает широкий спектр устройств и приложений для мониторинга физической активности, сна, питания и других аспектов здоровья. Они используют геймификацию для стимулирования пользователей к достижению своих целей и поддержанию активного образа жизни

#### 3.1.1.1 Вход пользователя в приложение
**Описание.** Работа в приложении возможна без создания профиля или же с авторизацией.

| Функция | Требования | 
|:---|:---|
| Вход без создания профиля | Приложение должно предоставить пользователю возможность войти в приложение анонимно |
| Регистрация нового пользователя | Приложение запрашивает имя и пароль. Пользователь вводит данные и регистрируется |
| Вход с авторизацией | Приложение запрашивает имя и пароль, после чего пользователь входит в профиль |

<a name="restrictions_and_exclusions"/>

### 3.1.2 Ограничения и исключения
